# learning-c-pointers
A beginner-friendly C program demonstrating pointers and pointer-to-pointer concepts with clean output and proper memory handling.

# C Pointers Fundamentals

This repository contains a simple and clean C program designed to explain
the fundamentals of pointers and pointer-to-pointer concepts.

## Concepts Covered
- Memory addresses
- Pointers
- Dereferencing
- Pointer to pointer
- Correct use of printf format specifiers
